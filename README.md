# ToDo App | Dani Dev

Aplicativo web simples para gerenciamento de tarefas, feito com **HTML, CSS e JavaScript**.


## Funcionalidades

- Adicionar tarefas novas
- Marcar tarefas como concluídas clicando sobre elas
- Remover tarefas
- Tabela organizada com status de cada tarefa
- Armazenamento local das tarefas usando **LocalStorage** (mantém os dados mesmo ao atualizar a página)


## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
